# Style Bible — Entrepreneur Quest art direction

## Fixed palette (HEX)
- Warm off-white: `#F5F0EB`
- Warm near-black: `#1A1614`
- Terracotta accent: `#C5380E`
- Amber accent: `#E6A817`
- Muted: `#7A7268`

## Anchor prompt frame (used in EVERY scene generation)
```
editorial 2D illustration, warm sunset palette (terracotta and amber on off-white),
melancholic but hopeful mood, soft film grain, cinematic composition,
NO people portraits — only environmental/POV/silhouette framing,
Instagram editorial style, premium magazine aesthetic
```

## Universal negative (appended to every prompt)
```
no smiling people, no handshakes, no laptops on desks, no team meetings,
no stock photo, no cozy lifestyle interior, no cartoon, no mascots,
no neon signs, no sunset beach, no pastel, no flat corporate illustration
```

## Marina rule
**NEVER** generate Marina's face. She is depicted as:
- Silhouette in background
- Over-the-shoulder POV
- Wide-shot of her environment (office, kitchen, room)
- Her hands in frame (partial)
- Never close-up portrait

## Consistency tricks (without character reference)
1. Same anchor phrase in every prompt
2. Same palette keywords in every prompt
3. Same lighting (warm afternoon/sunset)
4. Same composition (rule of thirds, low-angle or POV)
5. Same "melancholic editorial" mood
